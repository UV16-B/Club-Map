# Карта детства
Данное веб-приложение для поиска детского дополнительного образования с отображением на карте. Оно позволяет фильтровать кружки по городу, району, возрастной группе, виду активности и стоимости, а также добавлять новые кружки через административную панель.
## Возможности проекта
* Поиск кружков происходит по:
  * городу;
  * району;
  * возрастной группе;
  * стоимости;
  * виду активности.
* Отображение кружков происходит на Яндекс.Картах;
* Включена административная панель для добавления кружков;
* PWA-поддержка в наличии;
* Адаптивный интерфейс для следующих устройств:
  * смартфоны;
  * планшеты;
  * десктопы;
  * Smart TV;
  * смарт-часы.
* REST API работает с данными.

## Рабочие страницы

<details><summary><b>Главная страница</b></summary>

---
На главной странице доступны следующие элементы:
* Несколько выпадающих списков:
  * города;
  * района города;
  * вида активности.
* Календарь и кнопка с функцией подсчета возрастного диапазона;
* Инпуты для фильтрации маскимальной и минимальной цены;
* Кнопки применения фильтров;
* Карта для визуального результата.

Пользователь может выбрать несколько фильтров одновременно, и после нажатия на кнопку фильтрации метки на картеобновляются.

Для просмотра информации о найденных кружках можно нажать на метке, чей балун имеет название кружка, ссылку на сайт (по возможности), адрес, стоимость и телефон.

В разметке для часов карта отсутствует, для простоты был внедрен модуль списка кружков с названием, адресом и телефоном.

Приложение можно скачать через браузер (поддержка PWA) и работать с ним оффлайн.

</details>

<details><summary><b>Страница для администратора</b></summary>

---
Для доступа к администрационной панели необходимо перейти по данной ссылке:
```bash
http://localhost:3000/admin
```
Администратор на данной странице имеет право:
* Добавлять новую информацию о кружке:
  * название;
  * город;
  * район;
  * активость;
  * цену;
  * адрес;
  * координаты (широту и долготу);
  * телефон;
  * ссылку на сайт (при наличии);
* Просматривать результаты на карте.

Все поля (кроме поля с ссылкой) необходимо заполнить, иначе создание кружка провалится. При успешном создании нового кружка карта обновляется.

</details>
